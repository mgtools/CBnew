# AI Tools & Integrations

This directory contains configurations and utilities for AI-powered biological analysis.

## Structure

- **llm-prompts/**: Optimized prompts for LLMs (Claude, GPT) for biology tasks
- **ml-models/**: Pre-trained and custom ML models for biological predictions
- **integrations/**: API clients and integration code for AI services

## Available Models & Services

### LLM Services
- Claude (Anthropic) - General biology problem-solving
- GPT-4 (OpenAI) - Code generation and analysis
- Local models - Privacy-focused alternatives

### ML Models
- ESMFold / OmegaFold - Protein structure prediction
- BioBERT - Biological text mining
- Custom models - Domain-specific tasks
